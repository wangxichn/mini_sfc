
# Mini-SFC：一个参考MANO架构的SFC嵌入算法仿真框架

## 版本注意

在Mini-SFC开发过程中我们通过更新版本进行了对仿真框架不同程度的修改，使用该框架时请注意不同版本及其所提供的功能的对应性（尤其是在论文中所使用的模型、指标、方法等），如需要过去版本请留意仓库的中提供的历史发行版本
