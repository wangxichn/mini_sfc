MiniSFC documentation
=====================

.. image:: ../figures/logo.png
   :alt: MiniSFC
   :width: 100px
   :height: 100px
   :align: left

Mini-SFC is a simulation framework designed for Service Function Chain (SFC) orchestration algorithms. It supports both purely numerical simulations built on discrete events and container-based simulations that mimic real-time conditions, making it particularly suitable for studying SFC deployment and migration issues based on MANO architecture.
